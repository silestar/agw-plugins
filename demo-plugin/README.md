# Demo Plugin

这是一个示例插件，用于演示 AGW 插件系统的安装和使用流程。

## 功能说明

此插件不包含任何实际功能，仅用于验证插件安装、启用、禁用和卸载流程是否正常工作。

## 安装

在 AGW 插件管理页面搜索"Demo Plugin"并安装，或从本仓库直接下载。

## 使用说明

安装后可在插件管理页面看到此插件的状态变化：
- **installed**：已安装，未启用
- **running**：已启用，正在运行
- **stopped**：已禁用
